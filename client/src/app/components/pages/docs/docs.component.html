Register as a User

UI registration functionality not yet implemented, please use the method below for registering yourself as a new user.
To register as a user, you must first visit the API Documentation page at /api/docs/{{ apiVersion }} (linked here).
There, you will you need to scroll down to the Users section and expand the /api/users/register section.
Then click the Try it out button.
You should then see a textbox with something like this in it:
{{ '{\r\n    "name":"string",\r\n    "email":"string",\r\n    "username":"string",\r\n    "password":"string"\r\n}' }}
This is known as JSON (JavaScript Object Notation).
Now just replace the other string parts with your own information (remember to NOT delete the "s).
Here's an example:
{{ '{\r\n    "name":"John Doe",\r\n    "email":"jdoe123@gmail.com",\r\n    "username":"john_doe",\r\n    "password":"12345678"\r\n}' }}
Then click the Execute button.
Done! You are now registered as a user!
You should be able to login using that information and have all the core Pokémon ROMs and the 2 ROM hacks (Pokémon Prism and Pokémon Ash Gray).
If you get any errors when registering, read what they are, and fix your data so that they match the registration criteria.

Application Development Information

Stack Info

This application uses the MEAN stack, which stands for MongoDB, Express.JS, Angular 2, and Node.JS.

MongoDB serves as the database for the entire application. It is a document-typed database.
Express.JS serves as the API for the entire application. API stands for Application Programmed Interface. It is the middleman between the website and the database, constantly sending data in between the two.
Angular 2 is the client. It represents the website. It is the very front of the application, the part that the user can see and interact with.
Node.JS is the runtime. It is responsible for making sure the whole application is running properly. The whole application responds to, and runs on Node.JS.

Development Environment Info

This entire web app was programmed in VS Code.
VS Code is a lightweight text editor designed primarly for web development but can also be used for other types of code projects.
This application was developed on a native Windows machine (Surface Pro 6), but can also be run in a development (and production) environment on a Mac device running OSX.
