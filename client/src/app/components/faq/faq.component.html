Frequently Asked Questions

ROMs

What is a ROM?

A ROM is a Read Only Memory. Think of it as a CD with a game on it (ex. GTA V for PS4) but have it be inside cartridge media.

How do I play a ROM on my computer?

In order to play a ROM in your computer, you need an emulator.
I recommend Retroarch.

Cookies

What are cookies?

Cookies are little bits of data that websites store in your browser. They could be used for a variety of reasons, and for a lot of things.

Why does this site use cookies?

This website uses cookies to store your JWT (JSON Web Token) for authentication. The JWT is a Bearer token that the API uses to verify that you are actually logged-in and authorized to access data such as ROMs.

Responsiveness

Is this website mobile-friendly?

Yes, this website will work on screens as small as 320px in height/width.

Does this website have a mobile version?

No, this app is all one version itself. It will transform into a mobile interface as its viewport gets smaller.
